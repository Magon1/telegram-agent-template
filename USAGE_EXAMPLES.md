# 사용 예시

## 매일 자동 (07:00 KST)
별도 입력 없이 텔레그램으로 통합 브리핑 도착:
- 오늘 일정
- Research 메일 요약
- 텔레그램 DM 우선순위

## 자주 쓰는 명령어

### 통합 브리핑
- `/morning` — 즉시 통합 브리핑 (수동)

### 텔레그램 정리
- `/dm` — 최근 24시간 우선순위 정리
- `/dm 6` — 최근 6시간만 (빠름)
- `/dm 48` — 최근 48시간

### 기억
- `/remember Backpack 매니저는 Trish` — 영구 저장
- `/memories` — 전체 기억 목록
- `/forget Backpack` — 키워드로 기억 삭제

### 문서
- 텔레그램에 PDF 드래그 → 자동 인덱싱
- `/sync` — Drive 'Knowledge Base' 폴더 동기화
- "IR덱에서 매출 목표 뭐였지?" — 자연어 검색

### 모델 선택
- 기본 — Haiku (빠름)
- `/think` 또는 `/sonnet` — 깊은 분석
- `/opus` — 가장 똑똑

### 기타
- `/reset` — 대화 기억 초기화
- `/help` — 도움말
- `/auth` — Google 재인증

## 자연어 사용 예시

### BD/영업 관리
- "오늘 답장해야 할 거 있어?"
- "Backpack 최근 어떻게 됐어?"
- "이번 주 미팅 정리해줘"

### 전략·분석
- "Terminal 출시 전략 검토해줘" (자동 Sonnet)
- "/think 1년 후 회사 포지셔닝"

### 정보 검색
- "리서치 메일에서 거시 뉴스 정리"
- "OKX 딜 메모 핵심 요약"

### 작업 수행
- "내일 3시에 'Lucas 미팅' 1시간 잡아"
- "#01-프로젝트 채널에 '잠시 후 통화 가능' 보내줘"

## 워크플로 팁

### 답장 후 재스캔
1. `/dm` → 🔴에 누군가 떠있음
2. 텔레그램 가서 답장
3. `/dm` 다시 → 그 사람 사라짐

### 새 정보 들어오면 자동 기억
대화 중 봇이 알아서 중요 정보 저장.
나중에 "그거 뭐였지?" → 자동 회상.
